支持的平台 [待校准@1387]

我们在三个平台上支持ROS 2 Beta 3: Ubuntu 16.04 (Xenial) 、macOS 10.12 (斯莱瑞) 和Windows 10。我们为所有3个平台提供二进制包和如何从源文件编译的说明 (见 install instructions as well as documentation )。 [待校准@1452]
