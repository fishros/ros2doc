设置区域设置 [待校准@5897]

确保你有一个支持 UTF-8 的区域。如果您处于最小环境中 (例如docker容器)，则区域设置可能是最小的，例如 POSIX 。我们使用以下设置进行测试。但是，如果您使用不同的UTF-8支持的区域设置，应该没问题。 [待校准@5898]

locale  # check for UTF-8

sudo apt update && sudo apt install locales
sudo locale-gen en_US en_US.UTF-8
sudo update-locale LC_ALL=en_US.UTF-8 LANG=en_US.UTF-8
export LANG=en_US.UTF-8

locale  # verify settings

设置源 [待校准@5966]

您需要将ROS 2 apt仓库添加到您的系统中。为此，首先使用如下apt授权我们的GPG密钥: [待校准@5900]

sudo apt update && sudo apt install curl gnupg2 lsb-release
sudo curl -sSL https://raw.githubusercontent.com/ros/rosdistro/master/ros.key  -o /usr/share/keyrings/ros-archive-keyring.gpg

然后将仓库添加到源文件列表中: [待校准@5901]

echo "deb [arch=$(dpkg --print-architecture) signed-by=/usr/share/keyrings/ros-archive-keyring.gpg] http://packages.ros.org/ros2/ubuntu $(source /etc/os-release && echo $UBUNTU_CODENAME) main" | sudo tee /etc/apt/sources.list.d/ros2.list > /dev/null

安装ROS 2包 [待校准@5967]

设置好仓库后，更新您的apt仓库缓存。 [待校准@5968]

sudo apt update

桌面安装 (推荐): ROS、RViz、例程、教程。 [待校准@5969]

sudo apt install ros-foxy-desktop

基于ROS的安装 (裸骨): 通信库、消息包、命令行工具。没有GUI工具。 [待校准@5970]

sudo apt install ros-foxy-ros-base

尝试一些例子 [待校准@5916]

如果你安装了上面的 ros-foxy-desktop ，你可以尝试一些例子。 [待校准@5971]

在一个终端中，源文件安装文件，然后运行c talker : [待校准@5917]

source /opt/ros/foxy/setup.bash
ros2 run demo_nodes_cpp talker

在另一个终端源文件中，安装文件，然后运行Python listener : [待校准@5918]

source /opt/ros/foxy/setup.bash
ros2 run demo_nodes_py listener

你应该看到 talker 说这是 Publishing 消息 listener 说 I heard 这些消息。这将验证capis和Python api是否正常工作。万岁! [待校准@5919]
