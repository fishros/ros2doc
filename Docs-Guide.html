文档指南

本指南列举了ROS 1和ROS 2文档资料网站。这有助于了解特定的资源文件在哪里，如何提问以及维护哪些网站。 [小鱼@44]

目录

  • ROS Answers

  • ROS Design

  • ROS Discourse

  • ROS Docs (本站)

  • ROS 改善（Enhancement）建议（Proposals）(REPs) [小鱼@61]

  • ROS Prerelease

ROS Answers

目的

一个与 Stack Exchange 相似的关于ROS 1和2 的问答社区网站。 [Alyssa@48]

参考文献

有关如何使用ROS答案，请参阅 Contact Page 。 [待校准@50]

ROS Design

目的

一个记录许多早期ROS 2设计决策的地方。 [待校准@52]

注释 [待校准@53]

在ROS设计中， ROS 改善（Enhancement）建议（Proposals）(REPs) [小鱼@61] 更合适作为新参考文献。 [Alyssa@54]

ROS Discourse

目的

为ROS社区提供一般性讨论和公告的论坛。 [待校准@56]

参考文献

有关ROS论述讨论的更多信息，请参见 Contact Page 。 [Alyssa@57]

ROS Docs (本站)

目的

  • ROS 1和ROS 2 API文档 [Alyssa@59]

  • ROS 2核心文档，如安装说明、教程、指南、路线图等。 [待校准@60]

ROS 改善（Enhancement）建议（Proposals）(REPs) [小鱼@61]

目的

放置ROS 1和ROS 2设计和惯例的地方。 [Alyssa@62]

注释 [待校准@53]

REPs优于 ROS Design ，因为他们有一个更成熟的审查过程。 [待校准@63]

ROS Index

目的

链接到其他信息的所有包的索引列表。 [待校准@65]

注释 [待校准@53]

ROS索引用途 [Alyssa@66]

  • 查看包支持哪些ROS发布 (ROS 1和ROS 2) [Alyssa@67]

  • 链接到包仓库、API文档或网站 [待校准@68]

  • 检查包的许可证、编译类型、维护者、状态和依赖项 [Alyssa@69]

  • 寻找有关 ROS Answers 的问题 [待校准@70]

你可以把ROS 2索引想象成类似于ROS包的 Python Package Index (PyPi) 。 [Alyssa@71]

ROS Prerelease

目的

一个网站，可帮助您在本地计算机上生成命令以模拟 ROS Buildfarm 。 [Alyssa@73]

注释 [待校准@53]

目前，该网站的前端仅显示ROS 1发布版本。 [Alyssa@74]

ROS Robots

目的

展示使用ROS 1或ROS 2搭建的机器人的地方。 [Alyssa@76]

注释 [待校准@53]

此页面上的机器人是社区贡献的，有关如何贡献机器人的说明，请参阅网站。 [待校准@77]

ROS Wiki

目的

ROS 1文档和用户可修改的内容。 [Alyssa@79]

注释 [待校准@53]

  • 我们没有将wiki用于ROS 2，因为在维护wiki时会产生开销。对ROS 2来说wiki的功能将被 ROS Docs (本站) 、 ROS Index 、 ROS Discourse 和 ROS Answers 所取代 [Alyssa@80]

  • 维基将一直有效，至少直到最后的ROS 1发行版停产。 [Alyssa@81]

ROS.org

目的

ROS 1和ROS 2产品登录页面，提供了ROS的高级描述以及到其他ROS站点的链接。 [小鱼@83]

已弃用

ROS 2 文档

ROS 2 docs托管发布至 (含) Galactic版本的API文档。从Humble版本开始，API文档将托管在 ROS Docs (本站) 上。 [Alyssa@86]
